# 🎓 CampusAI — IIT Roorkee Unified Campus Intelligence Dashboard

A unified web dashboard for IIT Roorkee students to access all campus data from one place.

## ✨ Features

- 📚 **Library** — Search IITR digital book catalog with department and type filters
- 🍽️ **Canteen Menu** — All 21 IITR bhawan canteens with official menu and prices  
- 📅 **Events** — Upcoming IITR events (Cognizance, E-Summit, COMET26) with registration links
- 🎓 **Timetable** — Department and year wise timetable with professor and room details
- 🤖 **AI Assistant** — Chat with CampusAI to get instant answers about campus

## 🏗️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 14, React, TypeScript, Tailwind CSS |
| AI | Claude (Anthropic API) |
| Backend | 4 independent MCP Servers (Node.js + Express) |
